CSS Snippet Cheatsheet

CSS Snippet Samples for Flexbox and Grid

Flexbox Row

Use these three properties to create a Flexbox row layout.

        .row {
            display: flex;
            flex-direction: row;
            flex-wrap: wrap;
        }

Flexbox Column

Use these three properties to create a Flexbox row layout.

            .row {
                display: flex;
                flex-direction: column;
                flex-wrap: wrap;
            }

CSS Grid Layout

            .grid {
